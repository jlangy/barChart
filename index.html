Bar Chart API

Introduction

This is a simple API to draw a bar chart onto a web page. To use, include the script.js file in your webpage. This file contains the function drawBarChart(element, options, data), which takes three arguments. The first argument will be the html element to draw the bar chart into. The second element will be a set of custom options, and the third argument is the data you want to graph. Examples of sample bar charts are listed at the bottom.

Options

The options argument will be an object specifying the properties of the bar chart. The options are listed below.

  • height

    The total height for the bar chart to occupy. Enter as a string with units, e.g height: '400px'

  • width

    The total width for the bar chart to occupy. Enter as a string with units, e.g width: '600px'.

  • title

    The title for the bar Chart, e.g title: 'My Chart'.

  • xAxisTitle

    The title for the x axis of the bar chart, e.g xAxisTitle: 'Domain'.

  • yAxisTitle

    The title for the y axis of the bar chart, e.g yAxisTitle: 'Range'.

  • titleFontSize

    The font size for the main title to be in, as a string with units, e.g titleFontSize: '1.5em'.

  • axisTitleFontSize

    The font size for the x and y axis titles, as a string with units, e.g axisTitleFontSize: '1.2em'

  • titleFontColour

    The colour for the title as a string, e.g. titleFontColour: 'rgb(20,40,20)'.

  • barBorders

    Will include borders surrounding the bars, including individual stacked bars, if set to "on". Borders will be excluded if left out. , e.g. barBorders: 'on'

  • fontSize

    Sets the font size for the values inside the bars, and the x and y axis labels, e.g. fontSize: '0.8em'

  • yAxisTicks

    Can be used to set custom values on the y-axis. Provide as an array of numbers, e.g. yAxisTicks: [1,2,3,4,5]. If left out, y-axis will take on default values.

  • barColour

    Sets the colour of bars for the graph. Any data without a specified bar colour will default to this colour. e.g. barColour: 'red'

  • labelColour

    Sets the colour of number values inside the bars for the graph. Any data without a specified label colour will default to this colour. e.g. barColour: 'red'

  • dataPosition

    Sets the position for the data to sit inside the bar. Can be set to "top", "center", or "bottom". If unset, will default to center. e.g. dataPosition: 'bottom'.

Data

The data parameter can be an array of numbers, arrays, objects, or a mixture. Single number entries will be graphed as a single bar of that height, with default background colour and label colour. Array entries must be an array of numbers. They will be graphed as a stacked bar containing those values. e.g [1,2,3,[4,5],6]

If entered as an object, it must have a data property containing either a number, or an array of numbers. It can also have the property colours, which will be an array of the colours for that bar. You can specify the colour for each value of a stacked bar, in order, or have a single array entry if using a single bar. The object can also have the property labelColours, which will specify the colour for each label of the bars in order. e.g [{data: [3,4], colours: ['red', 'blue'], labelColours: ['blue', 'red']}, {data: 4, colours: ['red']}, {data:[4,5]}]

Examples

Calls drawBarChart(data, options, element) with parameters:

data = [3,2,5,4,3] ,

options = {
height: '400px',
width: '500px',
title: 'Simple Chart'
}

Calls drawBarChart(data, options, element) with parameters

data = [[3,4], [1,2,1,4], 12, [3,2]],

options = {
height: '400px',
width: '500px',
barBorders: 'on',
title: "Stacked Chart",
titleFontSize: '2em',
xAxisTitle: 'Domain',
yAxisTitle: 'Range',
axisTitleFontSize: '1em',
fontSize: '1em',
barColour: 'red'
}

Calls drawBarChart(data, options, element) with parameters

data = [{data: [4,3],
colours: ['red', 'cyan'],
labelColours: ['white', 'black'],
label: '2019'},
{data: [2,1,2,1],
colours: ['red', 'cyan','red', 'cyan'],
labelColours: ['white', 'black','white', 'black'],
label: '2020'}] ,

options = {
height: '400px',
width: '500px',
title: "Stacked Coloured Chart",
titleFontSize: '2em',
xAxisTitle: 'Domain',
yAxisTitle: 'Range',
axisTitleFontSize: '1.3em',
fontSize: '.8em',
yAxisTicks: [1,2,3,4,5,6,7,8],
barSpacing: 0.5 }
